Essential Windows Security Guide

Windows Security: Your Built-in Defender

Windows Security is a free, built-in tool that provides comprehensive protection for your Windows machine. It offers:

  • Anti-virus and threat protection
  • Firewall
  • Device protection
  • Performance monitoring
  • A few other services
  • If suspicious, download free MalwareBytes, uninstall it after checking your system

My Windows Defender Usage

I’ve done this safely on multiple windows machines for years. Once in a while, I will download something that tags as a problem. I trust Defender knows what it is doing. There are a few items that I keep which cause Defender to get upset. These are special geek tools which the normal person should not be using. I inform Defender to ignore items in those folders.

Several times when I suspected a problem, I would download MalwareBytes. It has not found anything that Defender missed, except once. That problem was not a downloaded program, it was something running in my browser from a webpage whos tab I left open. It was utilizing my GPU to mine bitcoins. I killed the browser and Bitwarden deleted a temporary file. it probably would have been deleted as soon as I closed the browser anyway. But I felt better!

Removing Pre-installed Antivirus

New machines often come with pre-installed antivirus software like Norton or McAfee. These are typically resource-intensive and free for a limited time, then they want money from you. It's recommended to remove these and activate Windows Security from Microsoft instead.

BitLocker: Encrypt Your Data

BitLocker encrypts your hard drive or SSD, securing your data even if the storage device is removed from your computer. To use BitLocker:

  • You'll need a Microsoft account
  • Encrypt your drive, save your recover codes somewhere else than your drive
  • Your data remains inaccessible without your password, even if the drive is removed from yoru machine

Account Security

Consider the following for your WIndows machine's security:

  • Create and use a local account for main work instead of a Microsoft account
  • Be cautious with biometric security like Windows Hello
  • Personally, I don't like biometrics, does your fingerprint work if you cut your hand? does you face work if you have a fever? If you are passout drunk, will you wake you the next day finding out someone used your face to log in and order 20 pizzas?
  • Will the government hold your phone or windows up to your face to log into your account and see what you're saying about the current administration?

Browser Security

Browsers are your gateway to the internet. To enhance browser security:

  1. Install the uBlock Origin extension to block ads and enhance privacy, even the NSA suggest using an ad blocker
    • Google Chrome and Chrome based browsers now block uBlock.
    • It still works on other browsers like Brave and Firefox
  2. Use multiple browsers for different purposes:
      Something like this:
    • Edge for Microsoft services
    • Chrome for chat applications
    • Vivaldi for Amazon
    • Brave for general browsing
  3. If you prefer using a single browser, consider Mullvad, Brave or Firefox. Change the default search engine to DuckDuckGo or some other privacy respecting search engine.
Security Tip: Be cautious of pop-ups requesting administrator permissions. Think twice before granting software permission to make changes to your system[1].

Useful Windows Tools

Everything
Free tool for extremely quick access to any file on your windows machine.
For example, list all 'jpg' in your 'download' folder, enter this: download jpg
or search your whole machine for 'Wednesday', just enter this: Wednesday
This tool is from voidtools website https://www.voidtools.com/downloads/

Photo Editors?
The more features a photo edit provides, the higher the learning curve. For simply editing the brightness or cropping a photo, the windows default editor work fine.
For more complex editing, I use "paint.net" https://www.getpaint.net/
For a professional editor, I recommend Gimp which is as good as Adobe Photoshop, free and has a high learning curve. https://www.gimp.org/

7-zip
FOSS ℹ️ Free and Open Source Software, maintain by real people that care ZIP format is free from Windows, but 7-Zip does everything ZIP does, and can access other files also. It tends to compress files better than Windows ZIP. And it's encryption includes the file names.
Windows and Linux versions
7-ZIP https://www.7-zip.org/download.html
PDF Reading
Edge, Chrome and other browsers can safely open a PDF file. Chrome will let you draw on it. Edge will let you draw, type and sign a PDF. Most people have little need to more complex PDF actions. Under no conditions would I install adobe reader.
BTW: you can save most web pages as a PDF file. You can view your bank's statement as a PDF and then save it.

Fixing Windows Problems

9 command line tools to fix Windows problems

FixMyWindows
These 9 microsoft tools can be used to fix windows problems. They are all command line tools, but you can find videos on how to use them.
See this video for details, this is the source of the list of tools, which also explains what they accomplish. https://www.youtube.com/watch?v=I-T9AjKQlt0
  • winget upgrade –all and winget upgrade
  • sfc /scannow
  • dism /Online /Cleanup-Image /RestoreHealth
  • chkdsk /r
  • ipconfig /flushdns ipconfig /release ipconfig /renew
  • netsh winsock reset
  • netsh int ip reset
    • netsh int space tcp reset
  • powershell "Get-PhysicalDisk | Format-Table FriendlyName, MediaType, HealthStatus, OperationalStatus"
  • tasklist taskkill /PID [number] /F taskkill /IM [processname.exe] /F
  • powercfg /energy
  • bonus memory test: mdsched

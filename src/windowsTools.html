Useful Windows Tools

Everything
Provides extremely quick access to all files on the windows machine for easy access.
For example, if you want a list of all jpg in yoru download folder, enter this: download jpg
Search your machine for Wednesday, enter: wednesday
This is from voidtools website https://www.voidtools.com/downloads/
WinDirStat
FOSS ℹ️ Free and Open Source Software tool to shows you where your files are stored in a grphical interface. This is so you can find where you have used up your drive.
From WinDirStat https://windirstat.net/
Photo Editor
The more features a photo edit provides, the higher the learning curve. For simply editing the brightness or chropping a photo, the windows default editor work fine.
For more complex editing, I use "paint.net" https://www.getpaint.net/
For a professional editor, I recommend Gimp which is as good as Adobe Photoshop, except it is free. https://www.gimp.org/
VSDC Video Editor
Free version for editing videos. And the $20 a year upgrade is worth it to do complex things
from Video Software Dev https://www.voidtools.com/downloads/
Veracrypt
FOSS ℹ️ Free and Open Source Software, maintain by real people that care about your privacy encryption program. Create volume on your drive or USB that only you or a real quantum computer can access. Find a video to show you how to use it.
Works on Windows and Linux
https://veracrypt.io/en/Downloads.html
LibreOffice
FOSS ℹ️ Free and Open Source Software, maintain by real people that care Microsoft Office alternative. This does over 99% of what Microsoft Word, Excel can do. It also contains other apps like LibreMath, and LibreDraw.
Works on Windows and Linux
LibreOffice Foundation https://www.libreoffice.org/
7-zip
FOSS ℹ️ Free and Open Source Software, maintain by real people that care ZIP format is free from Windows, but 7-Zip does everything ZIP does, and can access other files also. It tends to compress files better than Windows ZIP. And it's encryption includes the file names.
Windows and Linux versions
7-ZIP https://www.7-zip.org/download.html
PDF
These days Edge, Chrome and most other browser can safely open a PDF file. Chrome will let you draw on it. Edge will let you draw, type and sign a PDF. Most people lave little need to mroe complex actions. Under no conditions would I install adobe reader.
BTW: you can save most web pages as a PDF file. You can view your bank's statement as a PDF and then save it.
